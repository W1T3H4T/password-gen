# password-gen
Generate one or more passwords.

## Usage
`password-gen <enter>`

`password-gen <password length> <count>`

    Example: create 20 passwords, each 18 characters

## Building
You'll need CMake, and a compatible G++ compiler.

    mkdir build
    cd build
    CMake ..
    make
